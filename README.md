# 🎵 Spotify Inspired Music Player

A simple Spotify-inspired Music Player built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* ▶️ Play Music
* ⏸️ Pause Music
* ⏭️ Next Song
* ⏮️ Previous Song
* 📊 Progress Bar
* 🎚️ Volume Control
* ⏱️ Current Time & Total Duration Display
* 🎨 Modern Black & Red UI Design
* 📱 Responsive Layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure
